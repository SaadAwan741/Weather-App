# 🌦️ Weather App

A simple and responsive weather application that allows users to check real-time weather conditions by entering a city name.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Displays temperature in Celsius  
- 💧 Shows humidity levels  
- 🌬️ Displays wind speed  
- 🌤️ Dynamic weather icons based on conditions  
- ⚡ Fast and responsive UI  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (ES6)  
- Fetch API  
- OpenWeatherMap API  

---

## ⚙️ How It Works

1. User enters a city name  
2. App sends request to OpenWeatherMap API  
3. Receives weather data (JSON format)  
4. Updates UI dynamically with:
   - Temperature  
   - City Name  
   - Humidity  
   - Wind Speed  
   - Weather Icon
